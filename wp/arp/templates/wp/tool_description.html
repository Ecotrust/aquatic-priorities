Description of the Watersheds Priorities Software

Purpose

This tool was designed as a web interface to solve the question, "Given a set of constraints, which subbasins should I focus on to maximize conservation objectives." Typically, solving these problems, compiling the data and analyzing it is prohibitively complex and too time-consuming for the majority of potential users. We have created a suite of GIS datasets and an analytical approach, packaged as web-based tool, to facilitate iterative and collaborative exploration of conservation objectives.

Architecture

The software was built on many successful and proven technologies including Google Earth, Marxan, Django and MarineMap. The source code is open source and is available to view or download at the watershed-priorites code site.
